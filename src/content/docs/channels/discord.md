## 概述

Discord 是社区和游戏玩家的首选平台，OpenClaw 支持完整的 Discord 集成。

## 快速接入

1. 前往 Discord Developer Portal 创建应用
2. 添加 Bot 并获取 Token
3. 邀请 Bot 到你的服务器

```bash
openclaw channels add discord --token YOUR_BOT_TOKEN
```

## 支持的功能

- 服务器和频道消息
- 私信
- 斜杠命令
- 内嵌消息
- 反应和线程
