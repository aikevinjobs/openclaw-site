## 什么是智能体

OpenClaw 的智能体不只是聊天机器人，它能：

- 理解上下文和意图
- 调用工具完成实际任务
- 记住对话历史
- 在多个平台之间协调

## 智能体架构

```
用户消息 → 消息路由 → AI 模型 → 工具调用 → 回复用户
                ↑                    ↓
            会话记忆 ←──── 工具执行结果
```

## 多智能体

你可以运行多个智能体，每个有不同的角色：

```bash
# 创建一个客服智能体
openclaw agent create --name customer-support \
  --prompt "你是一个专业的客服，帮助用户解决问题"

# 创建一个翻译智能体
openclaw agent create --name translator \
  --prompt "你是一个翻译专家，将消息在中英文之间翻译"
```

## 会话记忆

智能体会自动记住对话上下文，支持：

- 短期记忆（当前对话）
- 长期记忆（跨对话持久化）
- 知识库检索
