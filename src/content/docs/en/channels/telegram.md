## The Easiest Way to Get Started

1. Find @BotFather on Telegram
2. Send `/newbot` to create a bot
3. Copy the Bot Token

```bash
openclaw channels add telegram --token YOUR_BOT_TOKEN
```

That's it! Send a message to your bot.

## Features

- Text, image, and file messages
- Voice messages
- Group chats
- Inline mode
- Command handling
