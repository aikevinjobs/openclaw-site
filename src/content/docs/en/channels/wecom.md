---
title: WeCom
description: Connect OpenClaw to WeCom (Enterprise WeChat)
---

## Setup

```bash
openclaw channels add wecom \
  --corp-id YOUR_CORP_ID \
  --agent-id YOUR_AGENT_ID \
  --secret YOUR_SECRET
```

## Features

- Direct and group messaging
- Group bot webhooks
- Event callbacks
- Organization structure queries
