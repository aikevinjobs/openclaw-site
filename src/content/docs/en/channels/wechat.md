## Overview

OpenClaw supports WeChat integration via Official Accounts and Mini Programs.

## Setup

```bash
openclaw channels add wechat \
  --app-id YOUR_APP_ID \
  --app-secret YOUR_APP_SECRET
```

## Features

- Text, image, and voice messages
- Customer service interface
- Template message push
- Mini Program integration
