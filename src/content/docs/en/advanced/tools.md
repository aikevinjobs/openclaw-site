## Built-in Tools

| Tool | Function |
|------|----------|
| Command Execution | Safely run commands |
| Web Search | Search real-time info |
| Weather | Get weather forecasts |
| Calendar | View and create events |
| Email | Read and send emails |
| File Processing | Read, write, convert files |

## Custom Skills

```bash
openclaw skill create my-skill
```

## Webhooks

```bash
openclaw webhook create --url https://your-service.com/hook
```
